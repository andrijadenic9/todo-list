# To-Do App

Welcome to the To-Do App repository! This application is designed to help users manage their tasks effectively. It's built using JavaScript to provide task management functionality, and task data is stored using the browser's localStorage.

## Features

- **Task Management**:
  - Add tasks that you need to complete.
  - Mark tasks as completed when you're done.
  - Edit tasks to make updates or changes.
  - Delete tasks that are no longer needed.

## View the App

Experience efficient task management with the To-Do App: [Access the To-Do App](https://andrijadenic9.github.io/todo-list/)
